Frontend development course

Front-end developer creates websites and web applications. His responsibilities include creating interface, and writing all the logic for user interaction with the site. For now At the moment, this is one of the most sought-after specialists with a low entry threshold.

What do we teach?

Html - site markup language. Css is a website styling language. Javascript - programming language, which is responsible for the logic on the site. For the most inquisitive, there is the possibility of further delving into the topic and learning about React.js. .

Your questions

What do you need to study? - Computer, internet access and VS code program. We last set in the first lesson.

What will my child know? - We are studying the entire main layer of languages and technologies to create a site. After the course, depending on the number of blocks passed, the student will have a set of modern and relevant knowledge on creating websites and web applications (HTML, CSS, JavaScript).

What is a web application? - This is a program that opens directly in the browser and does not require downloading to a computer.
